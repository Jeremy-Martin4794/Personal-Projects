Some basic Html syntax

<A href="something.html">
-creates a link to another file (EX: html, javascript, css)
-make sure file is in same directory or include file path

<link rel="stylesheet" href="something.css">
link to an external style sheet
-"rel" is relation between tag and href
-"href" (hypertext reference) is source file

<head>
the head is where metadata is stored (metadata is not displayed)
can use <title>, <meta>, <style>, and <link> tags in the head

<body>
the body contains paragraphs, headings, hyperlinks, images, etc.
can only be one body in an html doc

<script>
used to define a JavaScript script
can point to external script file with <script src="something.js">

<img src="url" align="left">
add an image somewhere on your website
can use different parameters such as: width="", height="", border=""

<h1>
heading one
this heading is most important (also has the biggest font)

<h2>
a heading with slightly smaller font (sub heading)
6 different types of headings (1 is largest, 6 is smallest)

<p>
tag for a paragraph
a paragraph always starts on a new line
can have <p1></p1>, <p2></p2>, <p3></p3>, etc.

<br>
tag for a line break

<div>
defines a division in html file
<div class="something"> (defines a class that can group data together to be styled in css or manipulated in JS)

<b>, <u>, <i>
tags indicate bolded, underlined, and italicised text respectively

<ul>
unordered list
use <li> for each list item

<ol>
ordered list
is numbered
use <li> for each list item

<dl>
description list
list an item with <dt> and descriptions underneath with <dd>

<table>
defines a table
use <tr> to define a table row
use <th> to define a table header
use <td> to define table data/cell
can use <th colspan="2"> for header to span 2 columns
can use <th rowspan="2"> for header to span 2 rows
add a caption to your table with <caption>
